Regular Expressions

You can declare regular expressions by using the RegExp constructor or just with the double slash (//) notation.

        var reg1 = RegExp("a");
        var reg2 = /a/;
  • The . character can represent any character.
  • To find ranges of characters, you can use the brackets "[]" notation 
                    var text = "Hola Mundo";
                text.match(/[a-z]/);
                // Finds the first character on the string that matches the a-z range
                // ouput: o
  • You can mix the ranges like [a-zA-Z0-9]

  • Sumarized

    • Word: \w is [a-zA-Z0-9], excluding the ñ and acuted characters
    • Decimal:\d is [0-9]

  • Controllers

    • i Insensitive
    • g All
    • m Multiline
    • + Repetitive structures
    • ? Matches any character not equal to the left character declared
    • * Repetitive structures
    • {2} Number of times that the character should appear sequentially
                    var text = "HolA MUndo \n Qué tal?";
                
                text.match(/[aeiou]|é/gi);
                // ouput: ["o", "A", "U", "o", "u", "a", "é"]
                
                text.match(/[aeiou]é/gi);
                // ouput: ["ué"]

                var text = "HolA MUndo \n Qué taal?";
                text.match(/a+/gi)
                // output: ["A", "aa"];

                var text = "Hoola";
                text.match(/o?/g)
                // output: ["", "o", "o" "", ""];

                var text = "Hoola";
                text.match(/o*/g)
                // output: ["", "oo", "", ""];

                var text = "Hoola o Mundo";
                text.match(/o{2,}/g)
                // output: ["oo", "oo"];
